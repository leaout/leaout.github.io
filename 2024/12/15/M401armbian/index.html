M401A 机顶盒刷机安装 Armbian
2024-12-15 ~ 2026-07-22
Linux Armbian 机顶盒

魔百盒 M401A 原本只是一款普通的机顶盒，但经过刷入开源操作系统 Armbian，它将获得全新的生命力和无限的可能性。本教程将一步步指导您完成整个刷机过程，从准备阶段到具体操作。

二、检查自己的 M401A 是哪种类型的版本

魔百盒怎么样刷系统 魔百盒311-1a YST 刷Armbian系统教程分享 上面的 GitHub issue 主要介绍了如何分辨 M401A 同型号但是不同结构。主要分为两种类型三种结构：

类型A

  • 强版本：2GHz 的 EMMC 参数和 1.8GHz 的 CPU 参数，dtb 驱动，ddr4 内存是 scy；emmc 标识为 sec 137 字样；板子的 logo 文字印刷在靠近蓝牙芯片处。
  • 弱版本：1GHz 的 EMMC 参数和 1.7GHz 的 CPU 参数，dtb 驱动，总体同上，emmc 标识为 silicongo。

类型B：ddr4 内存是 CXMT；emmc 标识为 silicongo 字样；板子的 logo 文字印刷在靠近 USB 接口处。

三、刷机前准备

开始刷机前需要准备一些工具：电视盒子、U 盘、键盘、显示器、HDMI 线。

1. 下载 Armbian 系统

https://github.com/ophub/amlogic-s9xxx-armbian/releases

进入上面的开源库，找到 Armbian_23.02.0_amlogic_s905l3a_jammy_6.1.18_server_2023.03.16.img.gz 类似格式的安装包，下载即可。

2. 将 Armbian 系统写入 U 盘

这里使用 Rufus 工具，下载地址：rufus-3.21 BETA.exe（提取码: kc09）

先选择 U 盘设备，选择刚刚下载的 Armbian 系统镜像，点击开始，U 盘会要求格式化，确定后自动写入 U 盘。

四、开始刷机

1. 盒子从 U 盘启动

拆开盒子，找到主板后面的重置按钮。插上 U 盘，先按住重置按钮，插上电源，过 5 秒左右松开按钮，等待看是否进入刷 Armbian 系统的流程。

2. 设置账户

进入 Armbian 系统后会加载一会程序，然后提示设置 root 密码和时区。

3. Armbian 写入 EMMC

输入命令：

提示选择安装设备型号，选择 306 M401A。FDTFILE 选择 meson-g12a-s905l3a-m401a.dtb。文件系统选择 1 (ext4)，接下来系统会自动进行初始化工作。

4. 重启系统

系统提示 Successful installed, please unplug the USB, re-insert the power supply to start the armbian. 表示已经安装完成，但此时不要拔 U 盘，输入 reboot 进行系统重启。

重启完成后可以断电拔出 U 盘，然后通电启动进入系统。
